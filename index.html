Features

Everything you need to manage your local development

YAML Pipelines

Define your CI/CD workflows using simple YAML files. Support for multiple steps, testing, building, and deployment.

Local Web Interface

Run from your browser. No external services or complex configuration required. Just Python and your code.

Project Templates

Generate base projects with complete structure, dependencies, and preconfigured pipeline for Flask, Django, and Node.js.

Logs & Reports

Visualize execution in real time with detailed logs and reports of each pipeline step.

Docker Support

Generated projects include Dockerfile and docker-compose for containerized development and testing.

Git Ready

Folder and file structure prepared for version control from the first commit.

Quick Start

Get started in under 2 minutes

1
Clone the repository
git clone https://github.com/HamiltonLeguizamon/LocalForge-Engine && cd LocalForge-Engine
2
Install dependencies
pip install -r requirements.txt
3
Run the interface
